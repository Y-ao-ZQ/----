## ⚠️ 重要变更

### 1. Jakarta EE 迁移

Spring Boot 3.x 使用 Jakarta EE 9+，所有 `javax.*` 包名改为 `jakarta.*`：

- `javax.persistence` → `jakarta.persistence`
- `javax.servlet` → `jakarta.servlet`

### 2. JWT API 变更

JWT 0.12.x 使用新的 API：

```java
// 旧版本
Jwts.parser().setSigningKey(secret).parseClaimsJws(token).getBody()

// 新版本
Jwts.parser()
    .verifyWith(key)
    .build()
    .parseSignedClaims(token)
    .getPayload()
```

### 3. Spring Boot 3.x 变更

- `spring-boot-starter-web` 不再包含 Hibernate Validator
- 需要单独添加 `spring-boot-starter-validation`
- MySQL Connector 包名变更：`mysql-connector-j`
