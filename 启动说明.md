# 校园二手交易平台 - 快速启动指南

## 🚨 重要提示

由于文件编码问题，后端 Controller 文件需要在 IDE 中手动修复。

## 解决方案

### 方法 1：使用 IntelliJ IDEA（推荐）

1. 用 IDEA 打开 `campus-secondhand-platform` 项目
2. 等待 IDEA 索引完成
3. IDEA 会自动检测编码问题并提示修复
4. 选择 "UTF-8" 编码保存所有文件
5. 运行 `CampusSecondhandPlatformApplication` 主类

### 方法 2：使用 VS Code

1. 用 VS Code 打开 `campus-secondhand-platform` 文件夹
2. 安装 "Java Extension Pack" 扩展
3. 打开每个 Controller 文件（在 `src/main/java/com/campus/secondhand/controller/` 目录下）
4. 点击右下角的编码显示，选择 "Save with Encoding" -> "UTF-8"
5. 保存所有文件后，运行主类

### 方法 3：手动修复编码（高级）

如果您熟悉命令行，可以使用以下命令批量转换：

```powershell
# 在 PowerShell 中执行
cd "d:\校园二手\campus-secondhand-platform\src\main\java\com\campus\secondhand\controller"
Get-ChildItem *.java | ForEach-Object {
    $content = Get-Content $_.FullName -Raw -Encoding Default
    [System.IO.File]::WriteAllText($_.FullName, $content, [System.Text.UTF8Encoding]::new($false))
}
```

## 启动步骤

### 1. 启动后端

```bash
cd "d:\校园二手\campus-secondhand-platform"
mvn spring-boot:run
```

启动成功后访问：http://localhost:8080/api

### 2. 启动前端

```bash
cd "d:\校园二手\campus-secondhand-platform-frontend"
PowerShell -ExecutionPolicy Bypass -Command "npm run dev"
```

启动成功后访问：http://localhost:3000
